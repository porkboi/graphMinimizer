Point Cloud Optimisation

ADMM hubs minimised against fit, path smoothness, shrinkage, graph variation, and station cost.

The browser runs a 2D, deterministic ADMM-style solver over hub locations x, z₁, z₂, z₃, u₁, u₂, u₃. Each frame updates the point assignments, shortest-path regulariser, quadratic shrinkage, graph total variation, and a station-count penalty.

f(x) = 1/2 Σᵢ Σᵧ∈I(xᵢ) ||y - xᵢ||²

g₁(z₁) = weighted path smoothness on shortest hub paths

g₂(z₂) = λ ||z₂||²

g₃(z₃) = μ Σ(i,j)∈E ||z₃,i - z₃,j||

g₄(x) = c_station · |stations|
